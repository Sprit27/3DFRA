Register

The register option is used to store the facial information of the target people, once the camera window opens press "space" to capture or press "exc" to exit

Start Session

The start session option is used to start the attendance session, it automaticly creates the table of current day, additionaly if the camera fails to recognis by chance the the attendance manager can use the unique number to manualy provide attendance

Record

Record displays the monthly count of attendance of each person

Clear

Clear option clears all the previous facial data, however the attendance data needs to be cleared manyaly
